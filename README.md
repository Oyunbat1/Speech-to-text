# Монгол хэлний хичээл хөрвүүлэгч

Монгол хэл дээрх хичээлийн бичлэгийг үг үсэг болгон хөрвүүлж, мөр бүрийг **Багш** эсвэл **Сурагч** гэж тэмдэглэсэн Excel файл үүсгэдэг энгийн вэб аппликейшн.

## Юу хийдэг вэ?

1. Та **багшийн дуу хоолойн жишээ** (зөвхөн багш ярьж буй 10-30 секундын богино бичлэг) оруулна.
2. **Хичээлийн видео/аудио** бичлэгээ оруулна.
3. Аппликейшн нь аудиог салгаж, `pyannote.audio`-р яригчдыг ялган, яригч бүрийг багшийн жишээтэй харьцуулж (яригчийн эмбеддингийн косинусын төстэй байдлаар), хэсэг тус бүрийг **Chimege** монгол хэлний STT API-аар хөрвүүлэн `.xlsx` файл буцаана.

Үр дүнгийн баганууд: `Start | End | Role | Text`. Хэрэв багшийг найдвартай таних боломжгүй бол хүснэгтийн дээд талд шар анхааруулга гарч, илүү удаан ярьсан яригчийг автоматаар Багш гэж тооцно.

## Тохируулга

### 1. Системийн шаардлага

- Python 3.10 ба түүнээс дээш
- `ffmpeg` PATH-д суулгасан байх
  - macOS: `brew install ffmpeg`
  - Ubuntu/Debian: `sudo apt install ffmpeg`
  - Windows: [ffmpeg.org](https://ffmpeg.org/download.html) сайтаас татаж, PATH-д нэмнэ

### 2. Python сангууд

```bash
python -m venv venv
# Windows: venv\Scripts\activate
source venv/bin/activate
pip install -r requirements.txt
```

### 3. HuggingFace (pyannote загварууд)

Pyannote нь HuggingFace-аас хязгаарлагдмал загваруудыг татаж авдаг. **Анх удаа ажиллуулахын өмнө**:

1. <https://huggingface.co> сайтад бүртгүүлнэ
2. Дараах гурван загварын хэрэглэгчийн нөхцөлийг зөвшөөрнө:
   - <https://huggingface.co/pyannote/speaker-diarization-3.1>
   - <https://huggingface.co/pyannote/segmentation-3.0>
   - <https://huggingface.co/pyannote/embedding>
3. <https://huggingface.co/settings/tokens> хаягт хандалтын токен үүсгэнэ

### 4. Chimege STT

<https://chimege.com/> сайтад бүртгүүлж API токенаа авна. Эндпоинтийн дэлгэрэнгүй мэдээллийг <https://docs.api.chimege.com/v1.2/en/> хуудаснаас үзнэ үү. `app/transcribe.py` доторх өгөгдмөл утгууд (URL, header) ихэвчлэн ажиллах ёстой ч шаардлагатай бол `.env` дотор `CHIMEGE_URL` / `CHIMEGE_CONTENT_TYPE`-ыг өөрчилж тохируулж болно.

### 5. `.env` тохируулах

```bash
cp .env.example .env
```

`CHIMEGE_TOKEN` болон `HF_TOKEN`-оо бөглөнө. Шаардлагатай бол `TEACHER_MATCH_THRESHOLD` (өгөгдмөл `0.5`) утгыг тааруулна.

### 6. Ажиллуулах

```bash
uvicorn app.main:app --reload
```

<http://localhost:8000> хаягийг нээж, хоёр файлаа оруулаад түр хүлээгээд `.xlsx` файлаа татаж авна.

## Гүйцэтгэл

Зөвхөн CPU дээр ажиллах ойролцоо хугацаа:

| Хичээлийн урт | Боловсруулах хугацаа |
| --- | --- |
| 5 мин | 3-6 мин |
| 30 мин | 25-35 мин |
| 60 мин | 45-70 мин |

GPU (CUDA) ашиглавал диаризац болон эмбеддинг тооцоолол 5-10 дахин хурдан болно. GPU байгаа бол CUDA-тай нийцтэй `torch` хувилбарыг суулгана уу.

## Мэдэгдэж буй хязгаарлалтууд

- Багшийн дуу хоолойн жишээ цэвэр (10-30 секунд, давхар яригч байхгүй) байх шаардлагатай.
- Зөвхөн монгол хэл. Англи хэл эсвэл хэл хольсон яриа алдаатай үр дүн өгч болзошгүй.
- Чимээ багасгах боловсруулалт хийдэггүй — шуугиантай ангийн бичлэг STT болон эмбеддингийн нарийвчлалыг муутгана.
- Сурагчдыг тус тусад нь ялгаж чаддаггүй — багш биш бүх хүн "Сурагч" гэж тэмдэглэгдэнэ.
- Синхрон хүсэлт — маш том файл хөтөч/прокси-н хугацаа дуусгах асуудалд орж болзошгүй. Дээд хязгаар нь 200 MB.
- Chimege дуудалт бүр төлбөртэй. Кэш хийдэггүй.

## Төслийн бүтэц

```
app/
  main.py        # FastAPI апп + замууд
  pipeline.py    # Бүхэл процессын зохицуулагч
  audio.py       # ffmpeg + pydub туслахууд
  diarize.py     # pyannote диаризац
  embed.py       # яригчийн эмбеддинг + жишээтэй харьцуулах
  transcribe.py  # Chimege STT клиент
  sheet.py       # openpyxl xlsx бичигч
static/
  index.html    # нэг хуудаст UI
tmp/
  uploads/       # ирж буй файлууд
  outputs/       # үүсгэсэн xlsx
```
